# Retro Paws Emporium - Selenium Test Automation

Automated test cases for Retro Paws Emporium web application using Selenium WebDriver and TestNG.

## Quick Start

1. Update `BASE_URL` in `RetroPawsTests.java` with your deployed app URL
2. Run tests: `mvn clean test`

## Test Cases Included

14 automated tests covering homepage, navigation, products, services, animals, contact form, cart, checkout, auth, admin, 404 page, responsiveness, and footer.

## Requirements

- Java 11+
- Maven 3.6+
- Chrome browser

## Running Tests

```bash
mvn clean test
```

Tests run in headless Chrome mode (CI/CD compatible).
